Motion & Icons

The motion and iconography spec. Durations, easing curves and the shimmer keyframe below are the canonical source — later component cards copy this code verbatim. Every line icon is drawn on the same 24px keyline at a 1.5px stroke so the set reads as one hand.

Easing curves

looping 3s comparison
--ease-outdecelerate · no overshoot
cubic-bezier(0.2, 0.8, 0.2, 1)
The workhorse. Every transition uses this unless a spring is explicitly called for.
--ease-springovershoot · settle
cubic-bezier(0.34, 1.4, 0.44, 1)
Reserved for celebration and confirmation — rank-ups, payout received. Note the overshoot past the edge.
--ease-out: cubic-bezier(0.2,0.8,0.2,1); /* decelerate to rest */ --ease-spring: cubic-bezier(0.34,1.4,0.44,1); /* y>1 → overshoot */

04

Reduced motion

honor prefers-reduced-motion always
✓ Do
Collapse animations to a static end state. The skeleton still reads as loading — it just holds still instead of sweeping.
✕ Don't
Never leave loops, spins or sweeps running unguarded. Motion-sensitive users get no relief and the page feels restless.
/* ship this guard on every card */ @media (prefers-reduced-motion: reduce) { *, *::before, *::after { animation-duration: 0.01ms !important; animation-iteration-count: 1 !important; transition-duration: 0.01ms !important; } }
