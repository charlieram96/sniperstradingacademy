Overlays

Everything that floats above the page — confirmation dialogs, a destructive alert, an anchored popover and a tooltip. All ride the --surface-overlay plane on --sh-lg over a scrim. Shown here as static panels inside labeled demo frames, with the scrim contained to each frame rather than the page.
